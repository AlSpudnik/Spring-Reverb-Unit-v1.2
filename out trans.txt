 Вихідний трансформатор драйвера ревербератора Tr1 виготовлений на залізі Ш 14 х 21 мм. 
F = 1.4см х 2.1см = 2.94см
F - Площа поперечного перерізу.

n1 = Q / F
n1 - кількість витків на 1v AC:
Q -  коефіцієнт діапазону частот (від 70 до 90).
n1 = 90 / 2.94 = 30.612 витків

 Визначимо з якою потужністю працюватиме пентод 6Ф5П при заданому режимі 238V 0.03A.
Pa = 238 * 0.03 = 7.14W
Pa - потужність на аноді лампи при заданому режимі.

  Будемо вважати вихідний імпеданс на аноді пентода 6Ф5П в данному режимі становить приблизно 6400 Ohm.
Up = sqr (Pa * Ra)
Up - Змінна напруга на аноді лампи.
Ra - вихідний імпеданс анода лампи.
Up = sqr (7.14 * 6400) = 46V AC

 Визначимо кількість витків первинної обмотки.
w1 = Up * n1
w1 - кількість витків первинної обмотки (primary winding).
w1 =  46 * 30.612 = 1408 витків

 Визначимо який потрібен коефіцієнт транcформації з навантаженням на вторинну обмотку, яке становить 8 Ohm impedance.
n = sqr (Ra / Rn) 
n - коефіцієнт трансформації (transformation ratio or turns ratio).
Ra - вихідний імпеданс анода лампи, або вхідний імпеданс трансформатора (primary impedance or input impedance). 
Rn - імпеданс гучномовця під'єднаного до вторинної обмотки або вихідний імпеданс трансформатора (secondary impedance or output impedance). 
n = sqr (6400 / 8) = 28,28

 Визначимо кількість витків вторинної обмотки.
w2 = w1 / n
w2 - кількість витків вторинної обмотки (secondary winding).
w2 = 1408 / 28,28 = 50 витків
------------------------------------------------------------

 Визначимо діаметр дроту для первинної обмотки.
d1 = 0.6 * sqr I
d1 - діаметр дроту для первинної обмотки (mm).
I - сила струму при заданому режимі роботи пентода (A).
d1 = 0.6 * sqr 0.03 = 0.1 mm
 
  Визначимо діаметр дроту для вторинної обмотки.
Is = sqr Pa / Rn
Is - максимальний струм у вторинній обмотці (A).
Is = sqr 7.14 / 8 =  0.945 A

d2 = 0.6 * sqr 0.945 = 0.58 mm
-------------------------------------------------------------

Первинна обмотка містить 1470 витків дроту діаметром 0,12 мм, вторинна 52 витки дроту діаметром 1 мм. Намотка здійснена у такій послідовності: спочатку намотана частина вторинної обмотки - 33 витки, поверх неї вся первинна - 1470 витків, потім домотана вторинна - 19 витків. Дві частини вторинної обмотки з'єднані послідовно.

    Перевіремо коефіцієнт транcформації зроблених обмоток:
   n = w1 / w2
   n = 1470 / 52 = 28.27

    Перевіряємо який вийде вхідний імпеданс трансформатора з навантаженням на вторинну обмотку 8 Ohm:
   Ra = Rn * (n * n)
   Ra = 8 * (28.27 * 28.27) = 6393.54 Ohm

    Перевіряємо вихідний імпеданс вторинної обмотки трансформатора:
   Rn = Ra / (n * n)
   Rn = 6393.54 / (28.27 * 28.27) = 8 Ohm
